HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • An element is two tags and everything between them.
  • h1-h6 are the different available levels of header importance.

CSS

  • Margin indicates space around outside of element
  • Padding indicates space around content inside of element
  • Hex color codes can be used to specify a precise color for borders, shadows, backgrounds, etc.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • cd followed by directory name to switch to a directory, cd .. to move back
  • Be cautious about what directory you are in when pushing, pulling, opening VS Code, etc.

JavaScript

  • Zero indexed. Arrays start with first item indexed as "0" and move up sequentially from there.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script. Top to bottom in JS.
  • An array allows a function to apply to more than one variable.
